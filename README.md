# Minigame

## Description

A minigame for SDL testing

## Key Features

* Is a solo player game.
* You have free move.
* You can shoot infinity bubbles as a bullet.
* The enemy is the jellyfish

## Controls

* Move: Up, Down, Left and Rigth arrows.

* Shoot: Space bar.

* Exit game: Esc.
